# Mockingbird Releases

This repository hosts Mockingbird's notarized macOS downloads and Sparkle
update feed. Its contents are maintained by the release process in the
[Mockingbird source repository](https://github.com/awsleiman-lab/mockingbird).

- [Download the latest release](https://github.com/awsleiman171/mockingbird-releases/releases/latest)
- [Browse the source](https://github.com/awsleiman-lab/mockingbird)
- [Report a bug or request a feature](https://github.com/awsleiman-lab/mockingbird/issues)
- [Report a vulnerability privately](https://github.com/awsleiman-lab/mockingbird/security)

Please do not open issues or pull requests here. Release artifacts are signed,
notarized, Sparkle-signed, and published as immutable GitHub releases.
